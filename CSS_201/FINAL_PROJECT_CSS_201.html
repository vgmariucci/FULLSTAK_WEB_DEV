OBJECTIVES

The objectives for this final project is to show as many as CSS tools learned from the ULTIMATE FULL STACK WEB DEV COURSE.

For this reason, the layout of the page was built using the CSS grid style. The page is organized in 4 main regions:

  1. Main Header Area
  2. Navigation Column Area
  3. Contents Area
  4. Footnotes Area

  • In the main header area we have the CSS-201 FINAL PROJECT text.
  • In the navigation columns there are links to get access to each CSS example demonstration.
  • In the contents area we place all examples and explanations like this one.

The footnotes area were reserved to placing some complementary information about the page and its developers.

ADVANCED SELECTORS

Advanced selectors are useful when we want to pass some style configurations for an entire group of elements.

There are many different advanced selectors in CSS. Some commonly used advanced selectors are:

  • Child selector: >
  • Json selector: +
  • Sibling selector: ~

PSEUDO SELECTORS

Pseudo selectors are used almost like the advanced selectors saw before.

The diffrence is that we can use the advanced selectors as before to pass a class to a group of elements, but also we can indicate using a pseudo selector what effects is allowed to the elements.

The pseudo selectors are used with the following sintax:
.class_name : pseudo_selector { attributes ... }
